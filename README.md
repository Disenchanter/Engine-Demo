# シンプル 3D レンダリング学習プロジェクト

このプロジェクトは C++ / OpenGL ベースの小さな描画フレームワークです。

## 主な機能
- `Phong` / `PBR` マテリアル対応
- `ECS` による `Transform` と `Mesh` 同期
- `ImGui` でシーンにオブジェクトを追加
- ボックスと球の簡易マテリアル差分表示

## ビルド
このプロジェクトは `CMake` + `Ninja` を使用します。

例:
1. Configure
2. Build
3. 実行

（環境に合わせて Visual Studio かコマンドラインで実行してください）

## 使い方
- アプリ起動後、`Scene Controls` からオブジェクトを追加できます。
- `Shape Type` で `Box` / `Sphere` を選択できます。
- `Clear Color` で背景色を変更できます。

## メモ
現在の PBR 設定は、見た目比較のために材質係数を使う簡易モードです。
